Features

Secure

Every transaction is secured with robust cryptography, distributed though the global peer-to-peer consensus network.

Private

CryoNote utilizes the cryptographic system to transfer funds without identifying information of each user becoming visible within the blockchain.

Untraceable

CryoNote uses ring signatures to make transactions untraceable, meaning its nearly impossible to use blockchain analysis to determine if funds have been spent.

Limited Supply

No more than 21 million CryoNote will ever exist within the blockchain.

Fast Syncing

CryoNote's peer-to-peer network allows for substantially fast blockchain syncing compared to that of Bitcoin.

ASIC Resistant

CryoNote utilizes the CryptoNight-Dark (0.5 MIB ScratchPad) PoW hashing algorithm making it resistant to ASIC based machines, and encourages decentralized mining.

Low Fees

CryoNote allows anyone to send fast, private transactions anywhere, to anyone around the world for nearly zero-fees.
